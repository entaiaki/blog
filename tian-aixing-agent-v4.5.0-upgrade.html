天爱星Agent v4.5.0

LightAgent-Live 架构重构：sherpa-onnx TTS 接口 + ChatViewModel 重写 + 立绘1 回归

2026-07-15

背景

v4.4.x 的 TTS 链路（Kokoro ONNX）与 Live2D 解耦不够干净。v4.5.0 按照《AI女友 + 桌宠项目改造任务书》启动 LightAgent-Live 重构，新增 com.entaiaki.lightagentlive 包，用事件驱动 + 生产者-消费者模式重新设计 ChatViewModel。

核心改动

1. TTS 接口重写

新增 TTSController 接口：

  • isReady: Boolean — 同步就绪状态
  • isSpeaking: StateFlow<Boolean> — UI 可观察播放状态
  • speak(text, onPlayback: (FloatArray, Int) -> Unit) — PCM 回调，不自己播放
  • suspend stopSpeaking() — 挂起等待完全停止
  • shutdown() — 资源释放

2. SherpaOnnxTTSController 实现

sherpa-onnx VITS 引擎接入（Apache 2.0，完全离线），配套 TtsSentenceSplitter 中英文分句工具。生成与播放解耦：engine.generate() → Channel → AudioTrack 循环消费。

3. ChatViewModel 重构

维度旧新
播放架构TTS 内部自己播放audioChannel 解耦（生成一句推一句，播放 loop 消费）
状态同步回调嵌套StateFlow 桥接 TTS → UI
打断stop() 同步stopSpeaking() suspend + cancelAndJoin
Emotion+LipSync无串联speakAndAnimate() 一体串联

4. 新模块一览

  • tts/ — TTSController、SherpaOnnxTTSController、TtsSentenceSplitter
  • live2d/ — Live2DController 接口、CubismWebLive2DController、Live2DWebView（stub）
  • agent/ — EmotionAnalyzer（stub）
  • service/ — FloatingPetService（stub）
  • ChatViewModel.kt — 调度中心
  • MainActivity.kt — TTS 在 IO 线程初始化
  • ui/ChatScreen.kt — Compose UI + isSpeaking 驱动停止按钮

5. 立绘1 回归

将之前移除的立绘1 16 张 padded JPG 重新加入 assets/characters/illust1/，使用英文情绪名映射（idle/happy/thinking/sad/angry/sleeping/sobbing/crying/depressed/distressed/drowsy/sweating/pained/disgusted/serious/wink）。

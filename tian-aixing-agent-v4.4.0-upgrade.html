天爱星Agent v4.4.0

新增完整音乐模块：自然语言控制播放 · 本地扫描 · MediaSession · 歌词 · NeriPlayer桥接

2026-06-24

🎯 动机

之前天爱星只能聊天、记笔记、设提醒、查天气。用户说「放首歌」她听不懂。

v4.4.0 参考两个开源音乐项目——NeriPlayer（Android 聚合播放器，多音源+无损下载）和 ECHO NEXT（桌面 Electron HiFi 播放器）——将音乐能力完整封入天爱星同一 APK，通过 PlannerAgent → MusicTool 工具调用链路，实现对音乐的自然语言控制。

🏗️ 架构

用户: "天爱星，放首周杰伦的晴天"
  → PlannerAgent 识别意图
  → TOOL: music_control   PARAMS: {"action":"play","query":"周杰伦 晴天"}
  → MusicTool.execute()
  → MusicRepository.playSearch()
  → LocalMusicSource.scan()        // 先查本地 MediaStore
  → [无结果] NeriPlayerBridge     // 降级到 NeriPlayer
  → MusicPlaybackService.play()   // 前台 Service 播放
  → StateFlow 更新 → UI 刷新

全进程内完成，不走 Intent 跳转，零延迟。
